EzyCad is a modern CAD (Computer-Aided Design) application designed for creating and manipulating 2D and 3D models. It leverages OpenGL, ImGui, and Open CASCADE Technology (OCCT) to provide a robust and interactive user interface for CAD operations.

Zombi GeoSim

Zombi GeoSim is a geospatial simulation tool powered by WebAssembly. It allows users to explore and simulate various geospatial scenarios directly in the browser for high performance and interactivity.


YouTube video of current native development version.

How to Use

  1. Click the link below to launch the Zombi GeoSim application.
  2. Be patient and allow the application time to load.
  3. Mouse scroll zooms in and out.
  4. Hold down the "alt" key and move the mouse to pan the view.
  5. Hold down the "shift" key and move the mouse to rotate the camera.
  6. Use the available tools to modify parameters, visualize data, and run simulations in real-time.
  7. For more advanced usage, refer to the webAsmPlay documentation.
